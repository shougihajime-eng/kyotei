## 1. 完了している機能 ✅

### コア (Vite + React)
- Vite ビルドパイプライン (CDN 全廃)
- Tailwind / Recharts / cheerio バンドル済み
- ErrorBoundary が必ずスタックトレースを表示 (白画面なし)
- localStorage は safeStorage 経由で全部 try/catch

### 4 タブ + 1 (5タブ)
- 🏠 ホーム — 結論カード + 直近結果 + 1週間損益 + ニュース
- 📋 一覧 — 連勝系 EV ランキング (S/A 評価のみ強調 + スマホカード)
- 📅 検証 — エア / リアル タブ + レース別カード一覧
- 📈 グラフ — エア / リアル / 比較 タブ + 累計収支 / 日別 / 回収率 / 券種別
- 💼 設定 — 資金 + 戦略 + エア/リアル切替

### 予想ロジック (5 因子 + 補正)
- ① 1号艇有利度 (重み 0.40 — 6号艇偏重対策で増強)
- ② モーター 2連率 (重み 0.18)
- ③ 展示タイム (重み 0.12)
- ④ スタート力 / 平均ST (重み 0.18)
- ⑤ オッズ妙味 / EV (重み 0.12)
- 1号艇に +0.05 ボーナス、6号艇に -0.08 ペナルティ
- 6号艇本命チェック: 強根拠 (展示◎/モーター高/勝率高/インドミナント低/荒れ判定/内側進入歴) を 2つ以上 満たさない場合は除外
- 直前補正: 部品交換 / チルト / 展示気配 / 当地適性 / 風向
- ニュース キーワードで会場単位の軽微補正

### 連勝系 4 券種専用
- 単勝 / 複勝 廃止
- 2連単 / 2連複 / 3連単 / 3連複 のみ
- 仮オッズ完全排除 — 実オッズ無ければ「オッズ取得不可」表示

### 戦略別の買い目
- 攻め (aggressive): 3連単 + 2連単 / 3-5 点
- バランス (balanced): 2連単 + 3連単 / 2-3 点
- 安全 (steady): 2連複 + 3連複 + 2連単 / 1-2 点

### 1号艇信頼度 (5 段階)
- イン逃げ濃厚 / 1号艇やや有利 / 1号艇不安あり / 荒れ注意 / イン崩壊警戒
- 部品交換 / チルト 1.5+ で下方修正

### 展開予想
- 逃げ / まくり / まくり差し / 荒れ / 混戦 / 標準 + 一言コメント

### 総合評価 ★ 1〜5
- maxEV から算出 + 荒れ期待度で下方修正
- 推奨アクション: 本線 / 本線/穴 / 穴狙い / 見送り

### 画面の 「ガーっ」 防止
- auto-save useEffect の依存を recommendations から racesSignature に変更 (循環ループ撃退)
- StrictMode を本番から外す

- Recharts は isAnimationActive=false
- BuyDecisionCard は React.memo
- refreshAll は try-catch-finally で setRefreshing(false) を保証

### Vercel Serverless Function

- /api/news (公式 RSS Atom)
- /api/racer (選手プロフィール)
- /api/health (動作確認)

### Header

- 大きい更新ボタン (44px / 右上 / 状態表示)
- 予想スタイル切替 (compact StyleSelector / AI 推奨 💡 マーク)
