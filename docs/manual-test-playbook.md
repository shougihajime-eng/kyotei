# 実機テストプレイブック (Round 42)

本番: https://kyotei-two.vercel.app/

PC + iPhone Safari + Android Chrome の **3 環境すべて** で同じ手順を踏み、各項目に ✅ / ⚠️ を付けてください。

---

## 0. 起動 (初回 + リロード)

| # | 操作 | 期待挙動 |
|---|---|---|
| 0-1 | URL を開く | オンボーディング表示 (初回のみ) → 完了後、ホーム画面 |
| 0-2 | F5 (PC) / 引き下げて再読込 (スマホ) | 設定 (スタイル / モード) が**保持**される |
| 0-3 | iPhone で notch/Dynamic Island | Header が隠れず、Safe Area 下に正しく配置 |
| 0-4 | Android Chrome で URL バー消える/出る | 100dvh で画面切れず |

| 1-6 | リロード | 最後に選んだモードが保持 |
