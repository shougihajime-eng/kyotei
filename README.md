# 競艇 期待値分析アシスタント (Kyotei EV Assistant)

「全レース予想AI」ではなく、**勝てる条件だけやる「買わないAI」**。
長期回収率を残すために、買うべきレースだけを厳選します。

## 特徴

- **GO / CAUTION / SKIP / WAIT** の4段階で判定。SKIP判定のレースは買い目候補が出ません
- **過剰人気・買ってはいけない本命** を自動検出
- **直前オッズ変動** で妙味消失を警告
- **地元・コース・水面特性** を補助補正に反映
- **仮想購入モード** で1週間テスト → 本運用判定
- **localStorage** に保存 (ユーザーごとに完全独立、サーバー送信なし)

## 思想

- 買うレースを増やすより **買わない精度** を上げる
- 的中率より **回収率** を最優先
- 妙味のない人気は絶対に買わない
- データが古いレースは確認待ち

## 禁止事項 (実装していません)

- 自動ログイン
- 自動購入
- パスワード保存
- 規約違反のスクレイピング

## 使い方

1. 初回アクセス時にガイドが3ステップで表示されます
2. ダッシュボード上部の「📍 今やること」を見る
3. GO のレースだけ確認 (SKIP/WAIT は買えません)
4. 結果を「📝 結果入力」タブに入れると AI が反省コメントを生成

## ライブラリ

CDN から動的読み込み (jsdelivr → unpkg → cdnjs の順でフォールバック):
- React 18
- Recharts 2
- Tailwind CSS
- Babel standalone (JSX変換)
